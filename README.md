# Banking

## Tech Stack Used:
- Kotlin
- Android Studio
- Room Database
- MVVM Architecture with Data binding and View Binding
- Glide – image library for android

## Tables Used (Room Database – SQLite based):
### Account:
- accountID
- customerID
- balance
- type

### Transaction
- transactionID
- accountID
- type
- isCredit
- date
- amt

## User
- name
- customerID
- address
- email
